Detailed Description

The Duration class.

Store durations only with minutes, but can be initialized with hours and minutes, and can be displayed in hours and minutes. WARNING: this class don't prevent the user from giving negative minutes in entry, the result would still be positive but unpredictable. Thus, we won't accept any minutes value over 59.
